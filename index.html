Projects

Mage Hand: VR Computer Networking Education Platform

Mage Hand is a VR networking simulation built in Unity that allows students to set up and visualize computer networks in an interactive 3D environment. Students can place, connect and configure devices. Instructors can create custom lab scenarios for students to follow and complete. The platform also enables instructors to observe student progress in real time.

Music Recommender

This project uses the Spotify API to analyze songs from a user's selected playlist, identifying their musical preferences based on attributes like tempo, energy, and valence. It then provides personalized recommendations, suggesting new songs that match the user's favorite types of music.
